# Grievance Portal Project

This is a full-stack grievance management system with separate frontend and backend components.

## Project Structure

```
├── grievance-backend/     # Node.js/Express backend
├── grievance-frontend/    # React/Vue/Angular frontend (to be added)
└── README.md             # This file
```
